# 前端后台管理系统模板 java780

本项目是基于 Vue3、Vite 和 TypeScript 技术栈构建的前端后台管理系统模版。以下是对项目的主要介绍：

## 技术栈

- **前端框架**：Vue3
- **路由管理**：vue-router
- **状态管理**：Pinia
- **构建工具**：Vite
- **编程语言**：TypeScript
- **UI组件库**：Element Plus
- **图表库**：ECharts
- **国际化方案**：vue-i18n
- **HTTP库**：Axios（已封装）
- **数据模拟**：Mock（模拟后端接口）

## 功能模块

- **用户管理**：对系统用户进行管理。
- **权限控制**：分配不同用户角色，进行权限控制。
- **数据统计**：使用 ECharts 实现数据的可视化。
- **国际化**：支持多语言界面，方便不同用户使用。

## 系统角色

- **超级管理员**：拥有系统最高权限，可以进行系统设置和用户权限管理。
- **普通用户**：进行日常的业务操作。

## 运行环境

- Node.js 版本需 >=12.0.0
- 支持的现代浏览器（Chrome、Firefox、Safari 等）

## 部署步骤

1. 安装 Node.js（确保版本 >=12.0.0）
2. 克隆或下载项目代码
3. 进入项目目录，运行 `npm install` 安装依赖
4. 运行 `npm run dev` 启动开发环境
5. 运行 `npm run build` 构建生产环境代码
